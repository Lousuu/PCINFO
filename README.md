# PCINFO / HardwareVision

HardwareVision 是一个基于 .NET 8 和 WPF 的 Windows 硬件监控与游戏性能记录应用。

## Source code

完整源码维护在：

- [`develop` 分支](https://github.com/Lousuu/PCINFO/tree/develop)
- [`codex/session-startup-hotplug-storage` 功能分支](https://github.com/Lousuu/PCINFO/tree/codex/session-startup-hotplug-storage)

## Releases

已发布版本：[GitHub Releases](https://github.com/Lousuu/PCINFO/releases)

## Branch policy

- `main`：项目介绍，只保留 `README.md`
- `develop`：长期源码集成分支
- feature/codex branches：功能开发与审查
- release tags：已发布版本源码

当前公开版本仍为 HardwareVision v0.1.7。

`main` 分支有意保持为 README-only，不代表源码被删除。
